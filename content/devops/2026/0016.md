# Helm2 vs Helm3
- Tiller removed in Helm3
- Helm2 Rollback : Only Diff Chart
- Helm3 Rollback : Diff Chart and K8s Resource

# Helm Components
- Helem is logging Secret
- Artifact Hub : Use Helm Chart Repository

# Helm Chart
- apiVersion : v1(Helm v2), v2(Helm v3)
- appVersion : Application Version
- version : Helm Chart Version
- dependencies : Other Chart Dependencies

# Command
- helm repo add <name> <url> : Add Helm Repo
- helm search <keyword> : Search Chart
- helm repo list : List Repo
- helm list : List Installed Release
- helm install <release name> <chart name> --namespace <namespace> : Install Chart
- helm uninstall <release name> --namespace <namespace> : Uninstall Chart
