# OS install

1. bootable usb creation (ubuntu example)
   - diskutil list
   - diskutil unmountDisk /dev/disk4
   - create bootable usb
      - if server : sudo dd if=/Users/zaimyon/winitech/ubuntu-22.04.5-live-server-amd64.iso of=/dev/rdisk4 bs=1m status=progress
      - if desktop : sudo dd if=/Users/zaimyon/winitech/ubuntu-22.04.5-desktop-amd64.iso of=/dev/rdisk4 bs=1m status=progress
   - diskutil eject /dev/disk4

# Minikube install

1. Docker install
   - https://docs.docker.com/engine/install/ubuntu/
   - user add : to run docker without sudo
    ```
    sudo usermod -aG docker $USER
    newgrp docker
    ``` 
2. Minikube install
   - https://minikube.sigs.k8s.io/docs/start/?arch=%2Flinux%2Fx86-64%2Fstable%2Fdebian+package

3. kubectl install
   - https://kubernetes.io/docs/tasks/tools/

4. Start minikube
   - ingress controller use
     ```
     minikube addons enable ingress
     ```
   - port forward : ```sudo vi /etc/systemd/system/k8s-forward.service```
     ```md
     [Unit]
     Description=Kubernetes Ingress Port Forwarding
     After=network.target
    
     [Service]
     ExecStart=/usr/local/bin/kubectl port-forward --kubeconfig=/home/wini/.kube/config --address 0.0.0.0 -n ingress-nginx service/ingress-nginx-controller 80:80
     Restart=always
     RestartSec=10
     User=root
    
     [Install]
     WantedBy=multi-user.target
     ```
     - sudo systemctl daemon-reload
     - sudo systemctl enable k8s-forward
     - sudo systemctl start k8s-forward
   - image push
     ```
     docker build -t spring-quick-$1:1.0 . -f demo-$1/Dockerfile
     minikube image load spring-quick-$1:1.0
     ```
     ex) https://github.com/zarate02/quickkube
